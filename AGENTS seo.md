## Mission

## Non-Negotiable Rules
- Preserve existing UI and design.
- Preserve existing routes.
- Do not remove components.
- Do not introduce breaking changes.
- Optimize one page at a time.
- Wait for approval before continuing to the next page.

## SEO Requirements
Every page must include:
- Unique title
- Unique meta description
- Canonical URL
- Open Graph tags
- Twitter Card tags
- Correct H1 hierarchy
- JSON-LD schema
- Internal links
- Optimized image ALT text
